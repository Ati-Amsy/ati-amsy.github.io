# ati-amsy.github.io
VJ napi jelszó

Android:
Nyisd meg Chrome böngészővel.
Kattints a jobb felső sarokban a három pontra (menü).
Válaszd a "Telepítés" vagy az "Hozzáadás a főképernyőhöz (Add to Home Screen)" opciót.
Ekkor a telefonod kirak egy ikont a kezdőképernyőre, ami pont úgy fog elindulni, mint egy rendes alkalmazás (nem látszik a böngésző címsora).

iPhon:
Safariban nyisd meg, és használd a "Hozzáadás a főképernyőhöz (Add to Home Screen)" funkciót. Olyan élmény lesz, mintha appot használnál.
